# Codetrack - A Simple Version Control System

Codetrack is a lightweight and straightforward version control system designed to introduce the basic concepts of version control. Built using Node.js, Codetrack focuses on essential functionalities such as tracking file changes, committing updates, and managing different file states. With core operations like initialization, file staging, committing changes, and viewing commit history, Codetrack provides a practical way to understand and work with version control principles.
